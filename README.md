# SLAUNCH
A Steam games launcher that runs in terminal for linux
, useless but fun :)

# Tutorial
1. Download steam-launch.sh and install-slaunch.sh (make sure they are in the same directory)
2. Run install-slaunch.sh
3. Open terminal and type "slaunch"


# New features as of update 1.0.3
1. New command added "slaunch -list"
- Displays all owned games, not just installed ones.
- You cannot launch a game from this section, it is only to display your owned games.

2. New command added: "slaunch -info"
- Displays your account info. (does not display login information)
